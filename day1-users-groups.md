# Linux Users & Groups

## Users
- Current user: whoami
- Show user ID: id
- Show all users: cat /etc/passwd

## Groups
- Show groups: groups
- Show current user's groups: groups username

## sudo
- Run command as root:
`bash
sudo apt update

## chown
- Change file owner
- sudo chown username file.txt

## find
- Find a file
- find . -name "test.txt"

## grep 
- Search text
- grep "hello" file.txt
